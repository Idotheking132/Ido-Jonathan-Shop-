🎯 מה צריך לעשות כדי להפעיל את החנות:
==========================================

📋 רשימת משימות:

1. ✅ הקוד מוכן - כל הקבצים נוצרו!

2. ⚙️ הגדרת Discord Application:
   - לך ל: https://discord.com/developers/applications
   - צור אפליקציה חדשה
   - העתק את Client ID ו-Client Secret
   - הוסף Redirect: http://localhost:3000/auth/callback
   - אפשר Intents: SERVER MEMBERS + PRESENCE
   - הזמן את הבוט לשרת (עם הרשאות Administrator)

3. 📝 ערוך את קובץ .env:
   - פתח את הקובץ .env
   - הזן את Client ID שהעתקת
   - הזן את Client Secret שהעתקת
   - הזן את Guild ID (מזהה השרת - לחץ ימין על השרת → Copy Server ID)
   - שנה את SESSION_SECRET למשהו אקראי

4. 🚀 הפעל את השרת:
   - לחץ פעמיים על start.bat
   או
   - פתח CMD/PowerShell והרץ: npm start

5. 🌐 פתח דפדפן:
   - גש ל: http://localhost:3000
   - התחבר עם Discord
   - אם יש לך רול Admin - תראה כפתור ניהול
